# Projeto Pizzaria

Este é um projeto de uma página web para uma pizzaria, desenvolvido para colocar em prática conceitos avançados de layout responsivo, manipulação de DOM com JavaScript e estilização detalhada.

---

## 🎨 Design

O projeto foi baseado em um protótipo desenvolvido no **Figma**. <br>
O objetivo é garantir a melhor experiência visual e de usabilidade para o usuário.

---

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do site.
* **CSS3:** Estilização, fontes personalizadas e layouts (Flexbox/Grid).
* **JavaScript:** Lógica de interatividade (como o sistema de pedidos ou navegação).
* **Figma:** Planejamento de interface e extração de assets (imagens e ícones).

---

## 📁 Estrutura de Arquivos

* `index.html`: Estrutura principal da página.
* `estilo.css`: Folha de estilo responsável pelo visual da pizzaria.
* `script.js`: Arquivo contendo a lógica de funcionamento do site.
* `.vscode/`: Configurações de ambiente para desenvolvimento.

---

Durante este projeto, pude aprofundar conhecimentos em:
- Integração entre Design e Código.
- Organização de arquivos em projetos Web.
- Manipulação de eventos em JavaScript.
